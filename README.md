# assistente-pessoal-estoico-contemporaneo
assistente pessoal estóico baseado nos ensinamentos estóicos de SENECA adaptados para a realidade atual


Objetivo: criar um assistente pessoal que operacionaliza a sabedoria estoica de Sêneca — especialmente nas Epistulae Morales ad Lucilium — para navegar os conflitos e incertezas da vida contemporânea.

**fontes:**
O original : Epistulae Morales ad Lucilium (Cartas a Lucílio)
